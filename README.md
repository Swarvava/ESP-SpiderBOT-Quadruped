## üìñ Description

ESP SpiderBOT is a four-legged robotic platform developed using MicroPython on ESP32.
Each leg contains two servo motors (hip + knee joint), enabling lifting, stepping, and propulsion.

The robot is controlled wirelessly using Bluetooth Low Energy (BLE UART Service) from a smartphone.

A biologically inspired creep gait pattern is implemented to maintain stability by moving one leg at a time while the other three support the body.
